# GitOps Demo - Manifests

This repository contains the Kubernetes manifests for the GitOps lab.

- `deployment.yaml`: Defines the Pod running the Node.js app.
- `service.yaml`: Routes traffic to the Pod.
- `ingress.yaml`: Exposes the Service via an Ingress controller.

Remember to replace `boychungtinh` in `deployment.yaml` with your actual Docker Hub username.
